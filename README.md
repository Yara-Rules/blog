# Yara-Rules Blog

## Adding new posts

Simply add a new file in `content/post/<name>.md` with the following content as template.

```
+++
title = "Website Redesign"

author = "Yara-Rules"
date = "2017-02-22"

draft = true

categories = ["news"]
tags = ["new", "web", "design"]
+++

<Insert your post content here>
```
